# Roadmap

## Phase 0: Concept Lock

- Confirm art direction
- Confirm MVP scope
- Confirm target platform
- Confirm first fish roster and tank list

## Phase 1: Prototype

- Build the main tank scene
- Add one editable tank layout
- Add basic fish movement
- Add shop data and item purchase flow
- Save state locally

## Phase 2: First Playable

- Add three tanks
- Add eight fish
- Add decoration placement
- Add tank stats and missions
- Add codex

## Phase 3: Polish

- Add layered visual effects
- Improve fish behavior variety
- Add transitions and feedback
- Improve balancing and progression pacing

## Phase 4: Expansion

- Add mutations
- Add event fish
- Add special decorations
- Add multi-tank progression
